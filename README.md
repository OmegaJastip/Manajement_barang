## 📱 Aplikasi Manajemen Barang

Aplikasi manajemen inventaris (inventory management) berbasis Flutter yang dirancang untuk membantu pengelolaan barang dagangan secara efisien. Aplikasi ini支持 Android, iOS, dan Desktop.

---

## 🚀 Fitur Utama

### 1. Manajemen Barang (Items)
- ✅ CRUD Lengkap (Create, Read, Update, Delete)
- ✅ Pencarian berdasarkan nama, kode, atau barcode
- ✅ Filter berdasarkan kategori dan status stok
- ✅ Pagination untuk performa optimal
- ✅ Detail barang dengan riwayat transaksi

### 2. Manajemen Kategori
- ✅ CRUD Kategori
- ✅ Pengelompokan barang berdasarkan kategori
- ✅ Statistik per kategori

### 3. Transaksi Stok
- ✅ Transaksi Masuk (Stock In)
- ✅ Transaksi Keluar (Stock Out)
- ✅ Riwayat transaksi lengkap
- ✅ Referensi dan catatan transaksi

### 4. Dashboard & Analytics
- ✅ Total nilai stok
- ✅ Jumlah barang dan kategori
- ✅ Grafik stok rendah
- ✅ Statistik transaksi

### 5. Laporan (Reports)
- ✅ Laporan Stok Barang
- ✅ Laporan Transaksi
- ✅ Export PDF dan Excel
- ✅ Filter tanggal dan kategori

### 6. Scan Barcode
- ✅ Pencarian barang via barcode
- ✅ Scan saat transaksi
- ✅ Generate barcode untuk printing

### 7. Export & Import Data
- ✅ Export Semua Data (Backup Lengkap)
- ✅ Export Barang ke Excel
- ✅ Export Kategori ke Excel
- ✅ Export Transaksi ke Excel
- ✅ Import Barang dari Excel
- ✅ Import Kategori dari Excel
- ✅ File disimpan di folder Downloads

### 8. Sinkronisasi Cloud
- ✅ Upload data ke Firebase Firestore
- ✅ Download data dari cloud
- ✅ Backup manual ke cloud
- ✅ Sinkronisasi Items, Categories, Locations, Transactions

### 9. Update Aplikasi
- ✅ Cek update otomatis
- ✅ Download APK dari GitHub Releases
- ✅ Install update dengan mudah

### 10. Print Barcode
- ✅ Generate barcode untuk barang
- ✅ Print via Bluetooth printer
- ✅ Custom label ukuran
